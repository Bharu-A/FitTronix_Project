General Theme

Your page has a cyber-futuristic fitness / sci-fi aesthetic. Key points:

Dark background: Mostly bg-gray-900 and bg-gray-800/30 (with opacity layers for depth).

Neon / gradient accents: Cyan, pink, purple — often in gradients (from-cyan-500 to-purple-600, from-purple-500 to-cyan-400).

Glassmorphism / frosted glass effects: backdrop-blur-md and semi-transparent backgrounds (bg-gray-800/50) for cards.

Glow effects on hover: hover:shadow with neon-colored shadows to enhance futuristic look.

Typography: Bold, large headings with gradient text (bg-clip-text text-transparent) for a cyber / high-tech feel.

2. Color Palette
Purpose	Tailwind / CSS used	Notes
Primary background	bg-gray-900	Dark main canvas for the page
Secondary sections	bg-gray-800/30, bg-gray-800/50	Semi-transparent for card layers and sections
Cyan neon	text-cyan-300, from-cyan-500, to-cyan-600	Used in buttons, progress bars, gradients
Pink neon	text-pink-300, from-pink-500, to-pink-500	Accent gradient, highlights, borders
Purple	text-purple-300, from-purple-500, to-purple-500	Secondary gradient for headings and hover effects
Green / yellow / red	For difficulty badges (bg-green-500/20, bg-yellow-500/20, bg-red-500/20)	Visual feedback on workout difficulty
Borders	border-cyan-500/20, border-pink-500/30, border-purple-500/30	Soft neon border highlights for cards
Error / alerts	bg-red-900/90, border-red-500, text-red-200	For toast/error messages
3. Typography & Text Styles

Headings: Large, bold, gradient text for high-tech feel.

text-5xl to text-7xl, font-bold, bg-clip-text, text-transparent, gradient


Subheadings & buttons: Bold, uppercase, clear readability.

Body text: Light gray (text-gray-300), cyan for highlights (text-cyan-200), pink or purple for accent text.

4. Buttons & Interactivity

Gradient buttons: bg-gradient-to-r from neon colors (cyan → purple, pink → purple).

Hover effects: Glow and scale transform for a cybernetic feel (scale-1.05, shadow-glow).

Rounded edges: rounded-lg for a soft futuristic style.

Shadow: Light neon glow shadows (shadow-lg shadow-cyan-500/30).

5. Cards / Sections

Backdrop blur: backdrop-blur-md + bg-gray-800/50 → glassy feel.

Borders: Semi-transparent neon borders for emphasis.

Spacing: Generous padding and margin for airy futuristic design.

Hover effects: Slight lift or shadow on hover using motion.div from Framer Motion.

6. Animations / Motion

framer-motion used for:

Fade in / slide in content (opacity, x, y changes)

Hover interactions (scale on hover)

Rotating motivational quotes (AnimatePresence)

This adds dynamic cyber-energy, matching the futuristic fitness theme.

7. Visual Summary

Overall Style: Dark mode + neon cyber accents + glass-like UI + motion animations.

Mood: High-energy, futuristic, immersive, motivational.

UI Principle: Dark background → neon highlights → glass cards → glowing buttons → animated transitions.
